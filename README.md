# 📅 Booking SaaS Api

**Booking SaaS** — это B2B-платформа для онлайн-записи и управления услугами, ориентированная на малый и средний бизнес (салоны, барбершопы, студии, частных специалистов и т.д.).

Проект позволяет бизнесам:
- принимать онлайн-записи 24/7,
- управлять услугами, расписанием и клиентской базой,
- отслеживать бронирования и загрузку,
- масштабироваться без ручной рутины.

Платформа также предусматривает пользовательский сценарий, где клиенты могут удобно записываться на услуги и управлять своими бронированиями.

---

## 🎯 Цели проекта

- Создать **удобный инструмент онлайн-записи** для бизнеса
- Сфокусироваться на **бизнес-процессах и управлении**, а не только на форме брони
- Обеспечить гибкую архитектуру для дальнейшего роста:
  - аналитика,
  - автоматизация,
  - интеграции,
  - AI-ассистенты (в перспективе)

---

## 👥 Основные роли

- **Business** — аккаунт компании (управление услугами, записями, клиентами)
- **Manager** — сотрудник бизнеса с доступом к управлению
- **User (Client)** — пользователь, который бронирует услуги

---

## 🧩 Ключевые возможности (MVP)

- Онлайн-бронирование услуг
- Управление услугами и расписанием
- Клиентская база бизнеса
- Система ролей и авторизации
- Поддержка мультиязычности
- Адаптация под мобильные устройства
